Planche is a mysql client tool runs on your wordpress.

== Description ==
Planche is MySQL GUI client tool. It runs on your wordpress. You can use this by one click.
Planche was made by javascript. But this tool will be run like a GUI application.

Support http tunneling.
Support multi connection.
Support multi query editor.
Support multi query result tab.

Just enjoy it!

== Installation ==
1. Upload contents of the directory to /wp-content/plugins/ (or use the automatic installer)
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Click planche in your admin left sidebar
4. Click \"Run planche\" button
